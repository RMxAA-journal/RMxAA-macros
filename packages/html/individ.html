RevMexAA LaTeX macros: download of individual files

This page has not yet been updated for version 3.0, sorry. From the README file: 

    All you need to use this package is the file `rmaa.cls' 

    However, you may find the Author Guide and sample documents instructive. 
    

    File Manifest for rmaa.tar:
    ---------------------------
    
    README	        this file
    rmaa.cls		LaTeX class file for typesetting RMAA articles 
    rmaacite.sty	auxilliary style file for semi-automatic citing 
    			(deprecated)
    rmuser.tex		Guide for Authors
    rmprocsamp.tex	annotated sample proceedings article
    rmprocsamp_fig1.ps	postscript figure included by rmprocsamp.tex
    rmprocsamp_fig2.ps	postscript figure included by rmprocsamp.tex
    rmprocsamp_fig3.ps	postscript figure included by rmprocsamp.tex
    rmpostsamp.tex	sample 1-page poster article
    rmtest.tex		another sample article
    rmeditor.tex	Guide for Editors (incomplete)
    rmplatesamp.tex	sample article with a plate 
    rmplatesamp_fig.ps	postscript figure included by rmplatesamp.tex
    GPL			Terms and conditions for copying, distribution
                        and modification
    OLDLATEX/		Directory containing 4 style files that will
			let you use the macros even if you have an 
			obsolete version of LaTeX

    File Manifest for rmaa_editor.tar:
    ----------------------------------

    All the above, plus

    rmbooksamp.tex	annotated sample proceedings volume  
    rmsummsamp.tex	sample conference summary  
    rmaa.ist     	MakeIndex style file 
    splitbook.sh	Utility shell script 
    fixbb.sh		Utility shell script 
    fixhtml.sh		Utility shell script 
    sanitizeps.sh	Utility shell script 
    rmsc.ps		PS file of logo for title page
    rmsc_blank.ps	Blank version of above
