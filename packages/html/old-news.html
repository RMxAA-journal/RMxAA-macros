RevMexAA LaTeX macros: Old News

  • (09 Feb 2004) Minor release 3.18. Fixed typo in documentation. Added a zip file for Windows users.
  • (08 Apr 2003) Minor release 3.17. Added appendices environment that does the right thing.
  • (07 Apr 2003) Minor release 3.16.
    • Added extra vertical space between authors and accepted dates in header of journal articles.
    • Newcommand \Color{}{} for typesetting photometric colors (e.g., B-V).
    • Added psfrag.sty to the OLDLATEX directory for those who don't have it on their system.
  • (25 Mar 2003) Minor release 3.15. Updated README file to add some more up-to-date usage instructions.
  • (06 Feb 2003) Bug fix release 3.14. Added option nonatbib to prevent loading of the natbib citation mechanism. This is necessary for backward compatibility with articles prepared using the old (deprecated) rmaacite.sty package.
  • (06 Feb 2003) Bug fix release 3.13. More corrections to splitbook.sh - only of interest to editors.
  • (04 Feb 2003) Bug fix release 3.12. Corrects some minor problems when producing whole issue.
  • (31 Jan 2003) Bug fix release 3.11. I had managed to break rmaacite.sty. Now I have fixed it again.
  • (24 Jan 2003) Bug fix release 3.10. This time I have really added the latest version of natbib.sty to OLDLATEX directory. Also tried to cut down the number of messages written to screen when compiling.
  • (23 Jan 2003) Bug fix release 3.09. Added latest version of natbib.sty to OLDLATEX directory.
  • (15 Jan 2003) Bug fix release 3.08. Fixes the \indexauthor command, which I managed to break in 3.07. It now has an optional argument for putting the author name but without any accents. This is so the alphabetization of the Author Index entries will work correctly.
  • (14 Jan 2003) New minor release 3.07. Added proof option, allowing marginal notes for inter-editor and editor-author communication.
  • (06 Jan 2003) New minor release 3.05. Large number of minor improvements. Finally, added support for natbib.
  • (22 Nov 2002) New minor release 3.03. Added \email command for typesetting email addresses so as to allow line breaks in the middle.
  • (2002/09/20) New minor release 3.02.
    • Added booktabs.sty and paralist.sty to OLDLATEX/ directory to allow use with obsolete versions of LaTeX. (Thanks to J. Ballet for pointing this out).
    • Added support for sideways tables (via Sidewaystable and Sidewaystable* environments - not documented yet).
    • User guide still hasn't been updated - please refer to the example files.
  • (02 May 2002) Finally, a new public release. 3.0 has quite a few changes and new features. See the example files for details. The user guide is now out-of-date. Editor's version coming soon.
  • (23 Jan 2001) 2.1 is a bug-fix release.
  • (30 Nov 2000) New major version (2.0) introduces a few ``improvements'' to the style of proceedings papers.
